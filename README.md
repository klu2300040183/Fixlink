Overview

FixLink is a web-based service repair platform that connects customers with technicians for various repair and maintenance services. The platform allows users to raise repair requests, upload issue images, track request status, provide ratings, and report suspicious behavior. It is built using HTML, CSS, and JavaScript with browser local storage for data persistence.

Features
Customer Module
Submit repair service requests.
Select service categories such as:
Electrician
Plumber
Technician
Carpenter
AC Repair
Cleaning
Upload photos of the issue.
Provide location details.
Track service request status.
Technician Dashboard
View all incoming repair requests.
Accept service jobs.
Mark jobs as completed.
View completed service history.
Rating & Feedback System
Customers can rate completed services from 1–5 stars.
Ratings are stored and displayed in service history.
Fraud Reporting
Customers can report suspicious technician behavior.
Fraud-reported requests are highlighted for visibility.
Data Persistence
Uses Local Storage to save:
Active requests
Completed service history
Ratings
Fraud reports
Responsive Design
Mobile-friendly layout using CSS Grid and responsive media queries.

Workflow
Customer submits a repair request.
Technician views available requests.
Technician accepts a job.
Technician marks the service as completed.
Customer provides a rating.
Completed requests move to service history.
Suspicious behavior can be reported through the fraud-reporting feature.

Project Goal

The objective of FixLink is to simplify the process of finding reliable repair services while improving transparency through ratings, service history, and fraud-reporting mechanisms.
